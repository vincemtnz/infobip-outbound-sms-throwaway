## Outbound SMS test

Testing Infobip outbound SMS with `notifyUrl`

## Requirements

1. [Bun](https://bun.sh)
2. Infobip credentials (API url and key)
3. A webhook endpoint. I used https://webhook.site

## Setup

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
